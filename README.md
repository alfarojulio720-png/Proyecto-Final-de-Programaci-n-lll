# Proyecto-Final-de-Programaci-n-lll
Este es el proyecto final la aplicación se llama: GeoFactura Analytics 

Esta aplicación muestra en un mapa interactivo las sucursales de distintas empresas en El Salvador, como Pollo Campestre y Pizza Hut.
Cada sucursal visualiza sus datos financieros mensuales, incluyendo ventas, compras, ganancia y porcentaje de rendimiento.
Los marcadores del mapa cambian de color según el desempeño: verde para ganancias y rojo para pérdidas.
La información proviene de una base de datos MySQL, es procesada por un backend construido con FastAPI y se muestra en el navegador mediante JavaScript y Leaflet.js, permitiendo una visualización clara y dinámica del rendimiento empresarial.
